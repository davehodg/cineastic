B) General Markup Info & Structure - top

Reviewer HTML use basic PHP functionality. This should make things easier for you because in the includes folder there is a header.php, footer.php, and sidebar.php. These files are pulled on to every page so you only need to make changes that show up on every page once.

The files are commented where the header, sidebar, and footer all begin and end.

There are a few classes and ID's that you need to know of in order to customize Reviewer HTML.

  1. Home Page Banner

    • In includes/header.php, line 112 you should see 

      <div id="header_banner" class="featured_banner_dark">
      Removing the "_dark" section will make the banner white instead of gray.

    • The banner and the footer also have a subtle pattern in the background. To change that, you can go to css/style.css line 18. It's commented there, but your options are bg_cross-stitch.png, bg_grid.png, bg_noise.png, bg_stripes.png, bg_thatch.png / There are also _dark versions of each, i.e. bg_crost-stitch_dark.png.

  2. Content Layout

    • The opening div in each page file has the id "main_content_alt" which adds the subtle sidebar break on the right side. If you'd like to get rid of this, simple change "main_content_alt" to "main_content". Make note that you need to do this in each file. There are also two different sidebar styles that can go along with this, stretched and block. to change these, simply go to includes/sidebar.php and change "sidebar_stretch" to "sidebar_block". If you use sidebar_block, you need to change the id="main_content_alt" to id="main_content" ON EVERY PAGE or it will not display correctly.

    • To move the sidebar to the left simply go to includes/header.php and on line 30 change the class "sidebar_right" to "sidebar_left".

C) Rating Posts - top

Anywhere you want to display a star rating, you need it nest it in the following structure: 

			<div class="rating">
				<div class="stars_5"></div>
			</div>

For the actual star numbers, change the number in the class="stars_5" to what you want. 1 = one star, 15 = 1½ stars, etc. as shown in the chart on the right.

To change the colors of the stars throughout your site, you can do so by simply replacing the color name on line 630 of css/style.css. Your options are yellow, orange, red, blue, and black.

D) The Footer - top

By default, the footer is laid out to have a thin bar of the menu on the left, and 3 divs floated next to it. You can easily lose the menu and have 4 divs floated by removing the class"footer_three_wide" from line 1 of includes/footer.php and, of course, removing the div with id="the_footer_menu" and adding a fourth div class="footer_widget".

E) Contact Form - top

The contact form is on page-contact.php. Open up that file and put your website's name and your email on lines 31 and 32, respectively. Other than that, the only other thing you may need to change is the action="page-contact.php" on line 65. If you change the name of the actual page-contact.php file, you must change it here too, otherwise your site will 404 when somebody fills out the contact form.

F) Extras - top

There are five extra little div classes you may want to use in Reviewer: Half Width, Quarter Width, Three Quarters Width, an Alert Box, and a Warning Box. To use them, simply drop the divs with the classes in:

  • <div class="half_wide"> This will be half the page width </div>
  • <div class="quarter_wide"> This will be one fourth the page width </div>
  • <div class="threequarter_wide"> This will be three fourths the page width </div>
  • <div class="alert"> This will be a yellow alert box </div>
  • <div class="warning"> This will be a red warning box </div>
